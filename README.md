# TaskFlow API

TaskFlow is a simple Task Management REST API built using ASP.NET Core and Clean Architecture principles.

This project was created to practice real-world backend development concepts including:

- Clean Architecture
- DDD structure
- Repository Pattern
- JWT Authentication
- Entity Framework Core
- RESTful APIs
- AutoMapper
- Exception Handling
- Authorization & Ownership Validation

---

# Features

## Authentication
- Register user
- Login user
- JWT Authentication
- Protected endpoints

## Projects
- Create project
- Get all projects
- Get project by id
- Update project
- Delete project
- Get project tasks

## Tasks
- Create task
- Get all tasks
- Filter tasks
- Update task
- Update task status
- Delete task

---

# Architecture

The solution follows Clean Architecture:
```txt
TaskFlow.API
TaskFlow.Application
TaskFlow.Domain
TaskFlow.Infrastructure
```

- ASP.NET Core 9
- Entity Framework Core

- JWT Bearer Authentication
